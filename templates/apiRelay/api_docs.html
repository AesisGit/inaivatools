Overview

Note: This documentatios is unfinished.

This is a simple API that returnes standardized telephone numbers for Sweden and Finland.


This API expects two parameters as inputs: country code and the and telephone numbers. Telephone numbers need to be a string where the numbers are divided with a comma sign. All fields are required. This API returns three parameters: response message, country code and fixed telephone numbers.

Inputs

This API has 2 required input values.

Input: country_code
Description: Country code
Valid range: SE or FI
Type: String
Required: Yes


Input: telephone_numbers
Description: Telepone numbers divided by a comma
Valid range: Telephone numbers
Type: String
Required: Yes

Example of succesfull API calls
